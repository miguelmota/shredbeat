# shredbeat

> Music beat reacts to your keyboard shredding speed

<img src="https://github.com/miguelmota/shredbeat/blob/master/screenshot.png?raw=true" width="600">

start

```bash
npm run start
```

start in development mode

```bash
npm run start:dev
```

watch and build browser files

```bash
npm run watch
```
